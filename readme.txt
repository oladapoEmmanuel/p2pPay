The Application performs basic banking operations. The functionalilies are as follows: 
1. Users can be added to the app.
2. Users can deposit money into the app.
3. Users can send money to other app users.
4. Users can check their balance in the app.
5. Users can transfer their money out of the app. 

-> The p2pclass.php is the file where the Class and member variables are defined.

-> The index.php file contains the code to add new users and its the home page.

-> The balance.php file contains the code to check balance.

-> The bank.php file contains code to perform transactions.

-> The send.php file contains code to send money to another user.

-> The transfer.php file contains code to transfer money out of the application.
